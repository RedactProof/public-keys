Usage

Verification certificates (verification.json) exported from RedactProof include an Ed25519 signature. Use this public key with Web Crypto API or any compatible library to verify certificate authenticity.

You can also use redactproof.com/verify for convenience—however, verification is designed to be performed independently using standard cryptographic tools. RedactProof provides the verification page as a tool, not as an authoritative source. The cryptographic proof is self-contained in your exported files.
